# Visual Studio Code

<section>

## Visual Studio Code – Shortcuts

* `⌘`+`O` = Open
* `⌘`+`B` = Show/Hide directories
* `⌘`+`⇧`+`P` = Quick Search
* `⌥`+`Z` = Toggle line wrapping
* `⌥`+`⇧`+`↓` = Duplicate row
* `⌘`+`⌥`+`↑` / `⌘`+`⌥`+`↓`  = Place additional cursors below / above
