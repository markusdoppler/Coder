## EcmaScript2015 – ES6

→ [Babel: Learn ES2015](https://babeljs.io/docs/en/learn)

- `let` and `const`
- `class`
- Arrow functions `(param) => { console.log(param); }`
- Template Strings `` `Template ${"string"}` ``
- Promises
- Async + Await
- Destructuring `let [a, b] = [1,2]`
- Default parameter values
- Array.find()
- Array.findIndex()
- Exponentiation (`**`) (ES 2016)
